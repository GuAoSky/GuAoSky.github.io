搭建博客总结

发表于 2019-04-27

本地部署

1.注册github账户（为博客托管准备）
2.Git Bash 进行账户认证
3.下载node.js（为Hexo提供支持）
4.命令行安装cnpm（npm install -g -cnpm –registry=https://registry.npm.taobao.org）
5.cnpm安装Hexo（cnpm install -g hexo-cli）

github部署

1.新建仓库（注意前部分为github用户名：githubname.github.io）
2.安装部署插件（cnmp install –save hexo-deployer-git）
3.设置博客文件下的_config.yml文件
4.部署到github（hexo d）

第一个博客

1.新建一个博客文件（mkdir blogname）
2.初始化一个博客（hexo init）
3.启动该博客文件（hexo s）
4.新建一篇文章（hexo n “博客名”）
4.清理（hexo clean）
5.生成（hexo generate）
6.重新启动（hexo s）
