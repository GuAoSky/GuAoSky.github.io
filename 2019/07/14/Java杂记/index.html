1.Java中equal与==的区别

1.1 ==：
    当变量为基本数据类型时，比较的是两边值的大小。
    当变量为引用类型时，比较的是其指向的地址值。
1.2 equal():
    当没有对equal()重写时，比较的是指向的地址值。
    当重写了equal()时，比较的是其指向的内容。例如(Double,Date,Integer,String)等包装类
    equal()不能用于基本的数据类型变量。
