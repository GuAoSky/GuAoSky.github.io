JDBC

发表于 2019-05-26 | 分类于 Java

1.概念：

JDBC(Java Database Connectivity) Java数据库连接，Java语言操作数据库
本质：定义了一套操作所有关系型数据库的接口，再由各个数据库厂商实现这套接口(数据驱动jar包)。

2.实现步骤：

2.1 导入驱动jar包
    将网上下载的驱动包(mysql-connector-java-8.0.11.jar)复制到项目的libs文件下后右击libs包设置Add as Library

2.2 注册驱动
    Class.forName("com.mysql.cj.jdbc.Driver");  //驱动名可在mysql-connector-java-8.0.11.jar/META-INF/services找到(MySQL 5 之后可省略此步骤)

2.3 获取数据库连接对象(Connection)
    Connection con = DriverManager.getConnection("jdbc:mysql://localhost:3306/dbtop?useSSL=false","user","pwd");

2.4 定义sql语句
    String sql = "update top_table set movScor = 9.8 where movRank=1";  //sql注入 a' or 'a' = 'a'

2.5 获取执行sql语句的对象(Statement)
    Statement stmt = con.createStatement();

2.6 执行sql,接收返回结果
    int count = stmt.executeUpdate(sql);   // 可用结果集ResultSet rs = stmt.executeUpdate(sql)接收

2.7 处理结果
    System.out.println(count);

2.8 释放资源
    stmt.close();  //如果stmt为null，则会引发空指针异常
    con.close();

3.各个成员详解：

3.1 DriverManager：驱动管理对象
    注册驱动，告诉程序该使用哪个数据库驱动，获取数据库连接。
    细节：如果连接的是本机MySQL服务器，并且MySQL默认端口为3306，则url可简写为"jdbc:mysql:///dbtop?useSSL=false"

3.2 Connection：数据库连接对象
    获取执行sql的对象
        Statement createStatement()
        PreparedStatement prepareStatement(String sql)
    管理事务
        开启事务：setAutoCommit(boolean autoCommit)  // 调用该方法设置参数为false，即开启其事务。
        提交事务：commit()
        回滚事务：rollback()

3.3 Statement：执行sql的对象
    执行sql
        boolean execute(String sql)：可以执行任意的sql（了解）
        int executeUpdate(String sql)：执行DML(insert,update,delect),执行DDL(create,alter,drop)语句。
        ResultSet executeQuery(String sql)：执行DQL(select)语句。

3.4 ResultSet：结果集对象
    next()：使游标向下移动一行(起始位置指向表的列名)
    getXxx(参数)：获取数据
        Xxx：代表数据类型   // 如：int getInt() , String getString()
        参数：
            int：代表列的编号，从1开始   // 如：getString(1)
            String：代表列名称。  // 如：getDouble("列名")
            例子：
                ResultSet rs = stmt.executeUpdate(sql)
                rs.next();
                int id = rs.getInt(1);
                String name = rs.getString("name")

3.5 PreparedStatement：执行sql的对象(用于解决sql注入问题)
    解决SQL注入问题（sql:select * from user where username = 'xxx' and password = 'a' or 'a' = 'a'）
        预编译SQL：参数使用?作为占位符，例如：select * from user where username = ? and password =?;
        获取执行sql语句的对象：PreparedStatement Connection.prepareStatement(String sql)
        给?赋值：
            方法：setXxxx(参数1, 参数2)
            参数1：?的位置编号从1开始
            参数2：?的值  
            例子：
                String sql = "insert into table value(null,?,?)";
                pstmt.setString(1,"小明");
                pstmt.setDouble(2,"2333");
                int count = pstmt.executeUpdate();
                System.out.println(count);
    后期都会使用PreparedStatement来完成CRUD（增删改查）的所有操作，且执行效率更高。
